# Tools

## Cheat Sheet

有很多包含縮寫或單字的字卡，裡面有這個縮寫或這個單字的中英文解釋，幫助我練習記住許多縮寫或是單字。

### 抽卡機

需要可以抽卡，然後可以點擊顯示答案。

![](https://i.imgur.com/rhf2hwE.png)

1. 單字本身
2. 發音(Google US)
3. 英文解釋
4. 中文解釋
5. 常見的別名
6. 看更多，可以點擊進去看這個單字的更多解釋。
7. 回去看上一張
8. 輸入數字，可以去看你想去的字卡。
9. 再抽一張
